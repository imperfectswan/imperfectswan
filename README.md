ʜɪ. ɪ'ᴍ **𝘛𝘪𝘭𝘭**. ɪ'ᴍ 6ᴛᴇᴇɴ, ᴀ ꜰɪᴄᴛɪᴏɴᴋɪɴ(ɪꜰ ʏᴏᴜ ᴅᴏɴ'ᴛ ᴋɴᴏᴡ ᴡʜᴀᴛ ᴛʜᴀᴛ ᴍᴇᴀɴꜱ, ꜱᴇᴀʀᴄʜ ɪᴛ ᴜᴘ.), ᴀɴᴅ ɪ'ᴍ ᴄᴇʀᴛᴀɪɴʟʏ ʙɪᴘᴏʟᴀʀ. ɪꜰ ɪ ᴄᴏᴍᴇ ᴏꜰꜰ ᴀꜱ ʀᴜᴅᴇ, ɪᴛ ɪꜱ ɴᴏᴛ ɪɴᴛᴇɴᴛɪᴏɴᴀʟ. ɪ ʜᴀᴠᴇ ꜰʀᴇ𝘘ᴜᴇɴᴛ ᴍᴏᴏᴅ ꜱᴡɪɴɢꜱ, ꜱᴏ ɪᴡᴄ. ɪ ᴅᴏɴ'ᴛ ᴡᴀɴᴛ ᴛᴏ ʜᴜʀᴛ ᴀɴʏᴏɴᴇ'ꜱ ꜰᴇᴇʟɪɴɢꜱ.
